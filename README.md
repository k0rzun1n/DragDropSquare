DragDropSquare
====================
Drag squares onto circles of the same color. 
Double-tap a square to change it's color. 
Circles change color after random amount of time.
Made with Unity.

[This game on GooglePlay.](https://play.google.com/store/apps/details?id=com.k0rzun1n.DragDropSquare&rdid=com.k0rzun1n.DragDropSquare)
